# Real Estate Landing Page

A modern luxury real estate landing page built with Next.js, Tailwind CSS, and deployed on Vercel.

## 🚀 Live Demo

https://real-estate-landing-two-beta.vercel.app/

---

## ✨ Features

- Modern responsive UI
- Sticky navigation bar
- Hero section with CTA buttons
- Tailwind CSS styling
- Component-based architecture
- Optimized with Next.js App Router
- Vercel deployment integration

---

## 🛠️ Tech Stack

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Vercel

---

## ⚡ Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
